Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes based on distance from the equator.

Specifically: I was looking at the relationship between the following four variables:

  • Latitude vs. Max Temperatures
  • Latitude vs. Humidity
  • Latitude vs. Cloudiness
  • Latitude vs. Wind Speed (mph)

To accomplish this analysis, I completed the following steps:

  1. Generated a list of 1422 unique cities from CitiPy.
  2. For each of those cities, I queried the OpenWeatherMap API to see if the city was in the OWM dataset.
  3. After checking all 1422 cities, I had results on 1244 cities.
  4. I plotted these results using MatPlotLib, one graph for each of the four questions.
  5. I also overlaid a linear regression line on each plot to visualize if there was a linear relationship between latitude and each variable.
